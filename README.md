# DESAFIO-QA-BEEDOO-2026

*Qual você acredita ser o objetivo da aplicação?* |
 Resposta: Na minha visão o objetivo dessa aplicação é verificar os melhores pensamentos análiticos e técnicos visando procurar um melhor desenvolvedor para a equipe e ver como ele age nessas situações.

 *Quais são os principais fluxos disponíveis* |
  Resposta: Listar Cursos, Cadastrar Cursos e Excluir Cursos

  *Quais pontos do sistema você considera mais críticos para teste?* |
   Resposta: Não é possível a exclusão do curso, quando você exclui, ele continua ali e apenas aparece que você excluiu. Outro problema que encontrei é que eu considero crítico é que, é possivel cadastrar um curso vazio, sem nenhuma informação.
